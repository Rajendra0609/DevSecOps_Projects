## Learning Path

### Phase 1: Web Stack Fundamentals
- Start with Project 1-4 to understand web stack implementation

### Phase 2: Database Management
- Project 5: MySQL database management
- Project 35: PostgreSQL Administration
- Project 36: PostgreSQL DevSecOps

### Phase 3: Containerization
- Project 20: Docker fundamentals
- Project 31: Docker Advanced

### Phase 4: Orchestration
- Project 21: Kubernetes from scratch
- Project 32: Kubernetes Advanced

### Phase 5: CI/CD with Jenkins
- **Jenkins Manual Pipeline**: Learn Jenkins fundamentals
- **Jenkins Pipeline as Code**: Automate with Jenkinsfile
- **Jenkins Advanced - Nodes & Agents**: Distributed builds
- **Jenkins Administration Guide**: Complete administration

### Phase 6: Infrastructure as Code
- Project 23: Ansible
- Project 24: Terraform
- Project 30: CloudFormation

### Phase 7: DevSecOps
- Project 25: Security in CI/CD
- Project 26: Vault
- Project 27: AWS IAM Security
- Project 36: PostgreSQL DevSecOps

### Phase 8: Serverless & Cloud Native
- Project 28: AWS Lambda
- Project 29: ECS/EKS Fargate

### Phase 9: Web Servers & Application Servers
- Project 33: Tomcat
- Project 34: NGINX

## Getting Started

1. **Prerequisites**

   - Basic understanding of Linux
   - Git and version control

2. **Recommended Tools**
   - AWS CLI
   - Docker Desktop
   - kubectl
   - Terraform
   - Ansible
   - Helm
